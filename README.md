🍔 Swiggy Sales Analysis Dashboard:-
An Excel-based interactive dashboard that provides a comprehensive analysis of Swiggy's sales data. This project helps visualize key business metrics and uncover trends across time, geography, and food preferences.

📊 Dashboard Highlights:-       
•	Monthly Sales Trend — Track how sales grow or decline month over month
•	Top 5 Cities by Sales — Identify the highest revenue-generating cities
•	Sales by Food Type — Understand which food categories drive the most orders
•	KPI Cards — Quick snapshot of Total Sales, Total Orders, and Average Ratings

🛠️ Tools Used:-
•	Microsoft Excel — Data cleaning, analysis, pivot tables & dashboard design

📁 Dataset Overview:-
The dataset includes key fields such as order date, city, food type, order value, and customer ratings.

🎯 Purpose:-
This project demonstrates the use of Excel as a powerful business intelligence tool to turn raw sales data into clear, actionable visual insights.
